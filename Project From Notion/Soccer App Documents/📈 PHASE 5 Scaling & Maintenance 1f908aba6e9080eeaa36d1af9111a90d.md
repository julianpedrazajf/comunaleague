**🌐 Web Support with React Native (Optional – Post MVP)**

If you want to expand the app to web browsers (great for organizers):

**Requirements**

- Install react-native-web
- Set up with Expo for Web or Next.js + react-native-web

**Recommendations**

- Start with admin and management screens
- Use responsive design from the beginning
- Split mobile/web views if needed for better UX

**Common Limitations**

- Some native components won’t work in web
- Performance may vary depending on browser and libraries

**Independent Web Alternative**

- Build a separate web version with React.js or Next.js
- Reuse the same backend and database/API
