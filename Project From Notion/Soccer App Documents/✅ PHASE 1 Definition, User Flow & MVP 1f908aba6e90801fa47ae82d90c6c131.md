# ✅ PHASE 1: Definition, User Flow & MVP

**🎯 Problem It Solves**

- Centralizes registration and organization of amateur soccer tournaments in Colombia.
- Allows both individual players and full teams to easily join tournaments.

## **🧩 MVP – Core Features (Final Version – Based on Figma Flow)**

1. **Guest Mode Home (Read-Only Navigation)**
    
    • Users can explore the home screen and scroll through content
    
    • All actions (buttons/taps) trigger a redirect to Login / Register
    
    • Promotes curiosity and familiarization before registration
    
2. **User Authentication & Onboarding**
    
    • Login with Email, Google, or Apple
    
    • Register via form
    
    • Role: Player (Organizer role in future phase)
    
3. **Extended Player Profile Setup**
    
    • Name, Last Name
    
    • Age
    
    • Country and City
    
    • Preferred Position (Goalkeeper, Defender, Center Back, Right Back, Left Back, Sweeper, Midfielder, Central Midfielder, Defensive Midfielder, Attacking Midfielder, Winger, Right Winger, Left Winger, Forward, Striker, Second Striker)
    
    • Preferred Foot (Left, Right, Both)
    
    • Height (cm or ft/in toggle)
    
    • Skill Level (Beginner, Intermediate, Advanced)
    
    • Favorite Team
    
    • Profile Picture
    
4. **Home (Full Access, Post-Login)**
    
    • Main navigation buttons:
    
    – My Team
    
    – Match Schedule
    
    – Inbox
    
    **• Embedded statistics section (scrollable, not clickable):**
    
    – View top scorers, rankings, recent results, featured players
    
    – Play Solo
    
    – Join Team
    
    – Create Team
    
    – Profile
    
5. **Team Management (My Team)**
    
    • View squad
    
    • Assign roles (captain, substitute)
    
    • Confirm attendance
    
6. **Match Schedule Viewer**
    
    • Calendar of upcoming matches
    
    • Game info: time, date, location, opponent
    
7. **Inbox (Team and System Messages)**
    
    • Receive updates from organizers
    
    • Notifications about games, team invites, changes
    
8. **One Game / Daily Match Mode**
    
    • Quick, solo-friendly match participation
    
    • Great for casual engagement
    
9. **Join a Team Flow**
    
    • Browse open teams
    
    • Request to join / receive invitations
    
    • Submit form and payment (if required)
    
10. **Create a Team Flow**
    
    • Name team, choose badge/logo
    
    • Select match format (5 or 11)
    
    • Add/invite players
    
    • Submit registration and payment
    
11. **Player Profile Management**
    
    • View/update user data
    
    • Access saved preferences
